# IY4101 OOP Shapes Assignment

This Java project is for IY4101 Object Oriented Programming Assignment 2.

The program manages a list of shapes using object-oriented programming concepts such as abstraction, inheritance, polymorphism and encapsulation.

The project includes:
- Coordinates class
- Abstract Shape class
- Rectangle, Square, Circle and Triangle subclasses
- ShapeList class
- ShapeManagement menu system
- UML design
- Testing evidence
